# APS 2026 GPS Tutorials

This repository contains the source for the website:

https://ai4eic.github.io/APS2026_GPS_tutorials

The website hosts the material for the **APS 2026 Global Physics Summit tutorials**, including notebooks, examples, and supporting resources.

## Repository structure

- `notebooks/` – tutorial notebooks and exercises
- `data/` – example datasets used in the tutorials
- `figures/` – images used in the documentation
- `docs/` – source files used to build the website

## Website

The website is automatically built and deployed via **GitHub Pages**.

➡️ https://ai4eic.github.io/APS2026_GPS_tutorials
